# 100xBot Community Hub

Welcome to the 100xBot Community Hub! This repository serves as the central place for users to:

- Report bugs and issues
- Request new features
- Ask questions and get support
- Discuss use cases and best practices
- Request LLM model support
- Suggest integrations with third-party tools

## What is 100xBot?

100xBot is an AI-powered browser assistant that brings multi-model LLM capabilities directly to your browser. It enables:

- Intelligent web automation
- Advanced data extraction and scraping
- Multi-step workflows with AI reasoning
- Support for multiple LLM providers (OpenAI, Anthropic, Google, Groq, and more)
- Browser-based agent execution

**Main Repository:** [100x-bot/agent4](https://github.com/100x-bot/agent4)

## How to Report Issues

We use GitHub Issues with structured templates to help you report problems effectively. Choose the appropriate template:

### Bug Reports
Found a bug? Use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.yml) template.

Include:
- Detailed description of the bug
- Steps to reproduce
- Browser and extension version
- Expected vs. actual behavior
- Relevant logs or screenshots

### Feature Requests
Have an idea for a new feature? Use the [Feature Request](.github/ISSUE_TEMPLATE/feature_request.yml) template.

Include:
- Problem you're trying to solve
- Proposed solution
- Use cases and examples
- Priority level

### LLM Model Support
Want support for a new AI model? Use the [Model Support Request](.github/ISSUE_TEMPLATE/model_support.yml) template.

Include:
- Model name and provider
- API documentation links
- Key capabilities
- Use cases in 100xBot

### Integrations
Need integration with a third-party service? Use the [Integration Request](.github/ISSUE_TEMPLATE/integration.yml) template.

Include:
- Service name and API details
- Desired workflow
- Use cases

### Documentation
Found issues with documentation? Use the [Documentation Improvement](.github/ISSUE_TEMPLATE/documentation.yml) template.

### Questions
Have a question or need help? Use the [Question/Support](.github/ISSUE_TEMPLATE/question.yml) template.
